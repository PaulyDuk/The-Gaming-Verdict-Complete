Generate Reviews Automatically

This tool will automatically generate reviews from popular games using the IGDB API. Reviews will include cover images, developer/publisher info, and AI-generated content.

Important Notes:
  • Processing Time: Each review takes time to generate (API calls, image uploads, AI content)
  • Web Interface Limit: Maximum 25 reviews via web to prevent timeouts
  • For Larger Batches: Use Django management command: python manage.py auto_generate_reviews --count 100
  • Dependencies: Requires IGDB API access, Cloudinary setup, and stable internet connection
